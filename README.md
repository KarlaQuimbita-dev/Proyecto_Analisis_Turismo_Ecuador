## Descripción
Proyecto de análisis de datos donde se realizó la limpieza, almacenamiento y visualización de información turística.

## Herramientas utilizadas
- KNIME: Limpieza y transformación de datos.
- SQL Server: Almacenamiento relacional y consultas.
- MongoDB: Almacenamiento NoSQL.
- Power BI: Creación de dashboard y análisis visual.

## Proceso realizado

### 1. Limpieza de datos
Se realizó limpieza de archivos originales mediante KNIME aplicando:
- Eliminación de duplicados.
- Tratamiento de valores faltantes.
- Normalización de datos.
- Corrección de formatos.

### 2. Base de datos SQL Server
Se creó la base TurismoDW para almacenar los datos procesados y realizar consultas.

### 3. Base de datos NoSQL
Se realizó la carga de información en MongoDB mediante documentos.

### 4. Dashboard Power BI
Se desarrolló un dashboard para analizar la información mediante gráficos e indicadores.
